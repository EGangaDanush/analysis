# analysis# Student Performance Analysis Using Pandas

## Project Overview

This project analyzes student academic performance using the Pandas library in Python. The application reads student data from a CSV file, calculates average marks, identifies the top-performing student, and generates statistical summaries of the dataset.

## Objective

To learn data analysis, data manipulation, and statistical operations using the Pandas library.

## Technologies Used

* Python 3.x
* Pandas
* Matplotlib
* CSV Files

## Features

* Read student data from CSV file
* Calculate average marks for each student
* Identify the highest-scoring student
* Generate descriptive statistics
* Create graphical visualizations
* Display overall class performance
